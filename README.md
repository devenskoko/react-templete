## Try it now!

### Clone to local

```bash
git clone https://github.com/devenskoko/react-templete
cd react-templete
yarn i
```

## Usage

### Development

Just run and visit http://localhost:8080

```bash
yarn run dev
```

### Build

To build the App, run

```bash
yarn run build
```

And you will see the generated file in `dist` that ready to be served.
